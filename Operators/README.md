# Operators
- Arithmetic
    - "+"
    - "-"	
    - "*"	
    - "/"	
    - "//"	
    - "%"	
    - "**"
- To compare (Karşılaştırma)
    - ">"
    - "<"
    - "=="
    - "!="
    - "<="
    - ">="
- Logical (Mantıksal)
    - and --> Returns True if both conditions are true.
    - or --> Returns True if at least one is true
    - not --> Gets the opposite of the condition
- Assignment (Atama)
    - "="
    - "+="
    - "-="
